# CDN

```jsx
// css
<link
  href="https://cdn.jsdelivr.net/gh/MEDICNC/design-system@main/apps/uikit/dist/css/uikit-core.min.css"
  rel="stylesheet"
/>
// js (body 닫는 태그 앞)
<script src="https://cdn.jsdelivr.net/gh/MEDICNC/design-system@main/apps/uikit/dist/js/uikit-core.min.js"></script>
// icon (body 닫는 태그 앞)
<script src="https://cdn.jsdelivr.net/gh/MEDICNC/design-system@main/apps/uikit/dist/js/uikit-icons.min.js"></script>

// reset.css (필요할 시)
<script src="https://cdn.jsdelivr.net/gh/MEDICNC/design-system@main/apps/uikit/dist/css/reset.min.css"></script>
```

# 포함된 라이브러리

- Jquery 3.7.1
- [Multiple Select](https://multiple-select.wenzhixin.net.cn/)

# 캐시 무효화

Purge CDN cache

https://www.jsdelivr.com/tools/purge 에 입력

```jsx
https://cdn.jsdelivr.net/gh/MEDICNC/design-system@main/apps/uikit/dist/css/uikit-core.min.css
https://cdn.jsdelivr.net/gh/MEDICNC/design-system@main/apps/uikit/dist/js/uikit-core.min.js
https://cdn.jsdelivr.net/gh/MEDICNC/design-system@main/apps/uikit/dist/js/uikit-icons.min.js
```
