# 버튼 유의사항

- default size : md
- 아이콘은 레티나 환경 화질을 고려해 비율을 2배로 해야한다. (1.5배 나 2.5배 x -> 이미지 깨짐)

  (아이콘 사이즈 피그마 기준 24px 24px)

  xxl 버튼 (아이콘 사이즈: 32px 32px) :
  `<span uk-icon="icon: check; ratio: 1.333"></span>`

- 아이콘 파일 네이밍: 피그마 export 할때 있는 네임 그대로
