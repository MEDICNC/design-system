# Confirm

<Preview>
    <Confirm />
</Preview>

```jsx
UIkit.modal
  .confirm({
    title: "제목",
    text: "Confirm 입니다.",
    confirmButtonText: "적용",
    cancelButtonText: "닫기",
  })
  .then(
    function () {
      console.log("확인 클릭");
    },
    function () {
      console.log("닫기 클릭");
    },
  );
```
