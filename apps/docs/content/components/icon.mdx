# 아이콘 등록방법

- C:\workspace\design-system\apps\uikit\src\images\icons 에 svg 파일 추가 후 pnpm icons 그리고 pnpm compile
- C:\workspace\design-system\apps\uikit\dist\js\uikit-icons.js 해당 파일에 나열되어 있는 네임으로 적용

- 아이콘 파일 사이즈: 피그마 기준 24px 24px
- 아이콘 파일 네이밍: 피그마 export 할때 있는 네임 그대로

badge 내 아이콘 (아이콘 사이즈: 16px 16px) :
